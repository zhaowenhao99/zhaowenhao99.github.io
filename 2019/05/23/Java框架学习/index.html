  • 框架
    • 软件的半成品，已经完成部分的功能
  • Hibernate框架
    • 持久层的ORM框架
  • ORM
    • Object Relational Mapping，对象关系映射，将对象与关系型数据库的表建立映射关系，操作对象就可以操作表
